# 笹木椿｜茶道家・Cultural Hospitality Designer — ポートフォリオサイト

一服の茶を、記憶に残るおもてなしへ。

福岡を拠点に活動する茶道家・笹木椿の公式サイトです。
裏千家茶道を基盤に、企業・経営者・ホテルのための茶道体験、ホスピタリティ研修、文化体験設計を提供しています。

## 構成

シンプルな静的サイト(HTML / CSS / JavaScript)です。ビルド不要でそのまま配信できます。

- `index.html` — 日本語トップ(Hero / What I Do / For Hotels / For Executives / Philosophy / About / Programs / Podcast / Contact)
- `en.html` — 英語ページ(ホテル・海外担当者向け)
- `css/style.css` — スタイル(生成り・深緑・墨色・竹色を基調、椿色を差し色に)
- `js/main.js` — ナビゲーションとスクロール演出
- `favicon.svg` — 椿のシンボルマーク
- `images/` — 本人提供の写真(プロフィール・手元・道具・茶花・露地・床の間)とPodcast「雑茶」のアートワーク

## ローカルで確認する

```sh
python3 -m http.server 8000
# → http://localhost:8000
```
