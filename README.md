# 笹木椿｜茶道家 — ポートフォリオサイト

茶道から、暮らしと仕事の所作を整える。

福岡を拠点に活動する茶道家・笹木椿の公式ポートフォリオサイトです。
裏千家 専任講師としての茶道経験を軸に、茶道体験・ワークショップ、企業向けプログラム、講演・執筆、Podcast「雑茶」などの活動を紹介しています。

## 構成

シンプルな静的サイト(HTML / CSS / JavaScript)です。ビルド不要でそのまま配信できます。

- `index.html` — 1ページ構成のLP(Hero / Concept / About / Philosophy / Activities / Podcast / Contact)
- `css/style.css` — スタイル(生成り・深緑・墨色・竹色を基調、椿色を差し色に)
- `js/main.js` — ナビゲーションとスクロール演出
- `favicon.svg` — 椿のシンボルマーク
- `images/` — 本人提供の写真(プロフィール・手元・道具・茶花・露地・床の間)とPodcast「雑茶」のアートワーク

## ローカルで確認する

```sh
python3 -m http.server 8000
# → http://localhost:8000
```
